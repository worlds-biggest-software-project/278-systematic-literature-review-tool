# Systematic Literature Review Tool — Feature & Functionality Survey

> Candidate #278 · Researched: 2026-05-03

## Solutions Analysed

Primary solutions: Covidence, DistillerSR, CADIMA, Rayyan (QCRI), ASReview, RevMan, Covidence Cochrane, Elicit (AI-native), Semantic Scholar.

## Cross-Cutting Features

**Must-have**: Search strategy documentation, screening workflow (title/abstract, full-text), conflict resolution, data extraction templates, bias assessment tools, PDF annotation, reference management, export to reporting templates.

**Differentiating**: AI-powered title/abstract screening, AI-assisted data extraction, automated bias assessment, natural language processing for question analysis, browser-based collaborative workflows, API integrations with publication databases.

**Underserved**: Automated conflict resolution, study quality meta-analysis, network meta-analysis visualization, living review updates, semantic search understanding intent, automated PRISMA compliance checking.

**AI-augmentation**: Intelligent paper screening using active learning, automated data extraction, NLP-based question analysis, automated study quality assessment, semantic clustering of similar studies, automated PRISMA checklist completion.

## Legal & IP Summary

Core review methodology is public (Cochrane standards). No identified patent encumbrances affecting functionality.

## Recommended Feature Scope

**Must-have (MVP)**
- Search strategy documentation with query history
- Screening workflow (title/abstract and full-text phases)
- Conflict management and resolution workflows
- Data extraction template customization
- PDF annotation and highlighting
- Bias assessment tool (Cochrane Risk of Bias)
- Export to PRISMA or other reporting formats
- User management and screening assignment
- Basic statistics and summary generation

**Should-have (v1.1)**
- AI-powered screening using active learning
- AI-assisted data extraction from PDF and text
- Risk of bias automation
- Meta-analysis calculation and visualization
- Collaborative workflow with real-time updates
- Integration with publication databases (PubMed, Scopus)
- Citation deduplication and matching
- Study quality meta-analysis
- Living review update workflows

**Nice-to-have (backlog)**
- Network meta-analysis visualization
- Semantic search understanding research intent
- Automated PRISMA compliance checking
- NLP-based question analysis and keyword suggestion
- Multi-language paper support and translation
- Bayesian and frequentist meta-analysis
- Subgroup analysis automation
- Publication bias detection
